# cost-aware-router — каскадная маршрутизация по цене

Платим за топ-модель только когда дешёвая не справилась: каскад от дешёвой к дорогой с проверкой качества (quality-gate).

## Как пользоваться

1. Задай лестницу моделей по возрастанию цены: `CASCADE = [("gemini-flash",1),("sonnet-4.6",10),("opus-4.8",30)]`.
2. Дай `answer_fn(model, request)` (в проде — вызов модели) и `quality_fn(request, answer)->[0..1]` (LLM-judge, урок 2.5).
3. Запусти каскад:

```bash
python -c "import cost_router as c; \
print(c.cascade('запрос', lambda m,r: f'[{m}]', lambda r,a: 0.9 if 'opus' in a else 0.4))"
```

4. Результат: `{model, answer, quality, cost}` — самая дешёвая модель, прошедшая gate.

## Когда применять

- Массовые/смешанные нагрузки, где часть запросов простая.
- Перед сменой «всё через Opus» на экономную схему — сравни с `always_top_cost()`.
