<!-- **Requires:** платный API-ключ <провайдер> — только если применимо -->

## PROBLEM
<конкретная боль, которую решает урок>

## CONCEPT
<интуиция + диаграмма ASCII/mermaid>

## BUILD IT
<механизм с нуля в `code/` (без фреймворков, офлайн на CPU); ссылка на файл>

```

## Чек-лист

- [ ] Все 6 шагов заполнены, MOTTO — одна строка.
- [ ] `code/` запускается офлайн без платных ключей; есть `test_*.py`, `pytest` зелёный.
- [ ] `outputs/` содержит обещанный артефакт.
- [ ] Блок «Материалы» — 2–4 **проверенные** ссылки.
- [ ] Отмечено `[x]` в `PROGRESS.md`; пересобран `site/` (`node site/build.js`).
