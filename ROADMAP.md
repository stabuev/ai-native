заканчивается наблюдаемым результатом, а отдельным артефактом — когда он полезен для
переноса. Полная программа по урокам — в [`README.md`](README.md),
статус — в [`PROGRESS.md`](PROGRESS.md), интерактивно — в [`site/`](site/).

## Конструктор урока

`MOTTO / PROBLEM / CONCEPT / BUILD IT / USE IT / SHIP IT / проверка / материалы` —
возможные элементы. Урок включает только целесообразные для своего outcome элементы.
