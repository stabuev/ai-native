# Урок 12.3 · Презентация и защита

**Фаза 12 — Capstone** · **Результат фазы:** Автоматизировать реальный процесс end-to-end и защитить решение с метриками эффекта.

> **MOTTO.** Защита — это цифры: насколько упало время и стоимость, выросло качество, какой ROI.

## PROBLEM

Работающее решение (12.2) ничего не доказывает без измерений. Защита Capstone — не рассказ «я сделал агента», а **эффект**: сравнение «до/после» по времени, качеству и стоимости и расчёт ROI. Без baseline из спеки (12.1) защищать нечего.

## CONCEPT

```
метрики «до» (baseline) vs «после» (решение):
   время   ↓  improvement%   (меньше = лучше)
   стоимость ↓ improvement%  (FinOps, Ф9–10)
   качество ↑ improvement%   (eval, Ф2.5/5.5)
        │
   ROI = (экономия + выгода) / затраты
        │
   презентация + чек-лист компетенций
```

Честная защита: реальные замеры, а не «кажется лучше»; учтённая стоимость решения; ограничения и риски (Фаза 11) проговорены.

## BUILD IT

Замеры «до/после» + ROI + чек-лист компетенций: [`code/metrics.py`](../code/metrics.py).

- `improvement(before, after, lower_is_better)` — % улучшения (время/стоимость vs качество);
- `roi(before_cost, after_cost, value_gained)` — окупаемость;
- `before_after(...)` — сводка по метрикам; `competency_report(...)` — готовность компетенций.

```bash
python code/metrics.py
pytest code -q
```

Демо: время −70%, стоимость −80%, качество +25.7%, ROI 5.5. Это и есть содержание защиты.

## USE IT

Демо и обратная связь:

- **Замерь до/после** на реальных кейсах: время (хронометраж), качество (eval-harness 2.5 / валидация 5.5), стоимость (телеметрия 10.3 / FinOps 9.4).
- **Посчитай ROI** «до/после» (FinOps for AI): экономия + выгода / затраты на решение.
- **Покажи демо** процесса и собери обратную связь; проговори ограничения и риски (Фаза 11).

## SHIP IT

**Артефакт:** Презентация + чек-лист компетенций → [`outputs/presentation-and-competencies.md`](../outputs/presentation-and-competencies.md)

Структура защиты (проблема → решение → метрики до/после → ROI → риски → демо) + чек-лист освоенных компетенций курса. Это финал курса: реальный процесс с доказанным эффектом.
