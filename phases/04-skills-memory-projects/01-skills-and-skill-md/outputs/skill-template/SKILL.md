## Как переиспользовать (инструкция к шаблону — удалить в реальном скилле)

1. Скопируй папку в `.claude/skills/<name>/` (проектные) или `~/.claude/skills/<name>/` (личные).
2. Заполни `name` (kebab-case) и **`description` с триггерами** — это главное: по нему агент выбирает скилл.
3. Тело (`body`) грузится только когда скилл выбран (progressive disclosure) — не раздувай.
4. Проверь: `ls .claude/skills/*/SKILL.md`. Кросс-платформенно: Claude Code / Cursor / Codex.
5. Хороший `description`: глагол действия + объект + 3–5 разных формулировок-триггеров.
