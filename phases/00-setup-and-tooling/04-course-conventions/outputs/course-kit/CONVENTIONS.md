# Соглашения курса (course-kit)

Build It / Use It / Ship It — конструктор учебного перехода, а не обязательное
оглавление. Используй этот файл вместе с `conventions.py`.

## Физическая структура

```text
phases/<PP>-<phase-slug>/<LL>-<lesson-slug>/
├── docs/ru.md   обязательный связный материал
├── code/        опционально: исполняемая практика + тесты
├── outputs/     опционально: переиспользуемый артефакт
└── quiz.json    опционально: диагностический квиз
```

`PP` — номер фазы, `LL` — порядковый номер урока внутри фазы. Для урока `3.2`
ожидается путь `phases/03-<phase>/02-<lesson>/`.

## Выбор элементов

Перед написанием реши для каждого элемента: **оставить / объединить / опустить**.

- `MOTTO` нужен, если одна фраза организует понимание.
- `PROBLEM` и `CONCEPT` получают отдельные секции, только если постановке и механизму
  действительно нужно отдельное место.
- `BUILD IT` нужен, если сборка делает механизм наблюдаемым или входит в outcome.
- `USE IT` нужен, если реальный инструмент даёт отдельный перенос, а не повторяет пример.
- `SHIP IT` нужен, если отдельный артефакт будет переиспользован.
- Ошибки, квиз и материалы получают отдельные блоки только при самостоятельной пользе.

Заголовки можно переименовать и объединить. Нельзя сохранять пустую секцию ради
симметрии с соседними уроками.

## Два слоя готовности

**Педагогическая готовность** означает, что студент без скрытых пререквизитов выполняет
наблюдаемое действие, проверяет результат и переносит его в новую задачу. Форма практики
и проверки следует из outcome.

**Структурная готовность** означает только, что существует заполненный `docs/ru.md` без
маркеров стаба. Код, артефакт, квиз и материалы не обязательны. Если они добавлены, то
должны быть рабочими и поддерживать результат урока.

## Автоматизация

```bash
python conventions.py
python -c "from conventions import validate_lesson as v; print(v('phases/01-demo'))"
```

`validate_lesson(path)` не требует стандартные секции и `outputs/`. Пустой список
означает «материал заполнен», но не доказывает его педагогическую достаточность или
целесообразность выбранной структуры.
