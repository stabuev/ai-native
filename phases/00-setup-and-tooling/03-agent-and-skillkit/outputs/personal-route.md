# Персональный маршрут AI Native

Это рабочий план, а не оценка профессионального уровня. Заполни его результатом
`find-your-level` и пересматривай после практики и контрольных фаз.

- **Дата диагностики:** `<YYYY-MM-DD>`
- **Сквозная задача из 0.1:** `<номер и краткое название / нет задачи для агента>`
- **Критерий успеха:** `<критерий из той же карточки / не применимо>`
- **Роль агента:** `<рабочий кандидат / только среда курса>`

Роль «только среда курса» — нормальный результат: она означает, что карта 0.1 не
требовала агента для выбранных рабочих задач. Не переписывай карту задним числом только
ради согласованности уроков.

## Как читать действия

- `focus` — пройти фазу полностью;
- `refresh` — быстро восстановить модель, затем выполнить практику и проверку;
- `challenge` — начать с выходной проверки и артефакта, вернувшись к урокам при пробеле.

Действия `skip` нет. Capstone всегда остаётся `focus`. Диапазоны времени — ориентир
для календаря, а не обещание скорости.

## Сценарные проверки

Заполняй раздел только для реально пройденных входных сценариев. Один вопрос на
компетенцию — слабое доказательство, поэтому сохраняй результат как направление для
маршрута, а не как процент владения темой.

| Фаза | Что проверялось | Что получилось | Как маршрут изменился |
|---:|---|---|---|
| `<1 / 4 / 8>` | `<компетенции>` | `<сильные стороны и пробелы>` | `<focus / refresh / challenge и почему>` |

`baseline_competencies`, если он создан диагностикой, оставь в маршруте отдельным
JSON-блоком. После контрольной фазы `check-understanding` использует его только для
ориентировочного сравнения с новыми сценариями, а не для заявления о точном приросте
навыка.

## Критерии готовности

- [ ] Агент повторно запускается в той же рабочей папке.
- [ ] `agent-check.md` содержит правильный путь.
- [ ] Сквозная задача и критерий совпадают с одной карточкой из карты 0.1.
- [ ] Роль агента не противоречит выбранному там способу работы.
- [ ] `find-your-level` и `check-understanding` лежат в каталоге выбранного агента.
- [ ] Для каждой фазы есть действие и понятная причина.
- [ ] `challenge` нигде не трактуется как автоматический пропуск.
- [ ] В маршруте нет точных часов, не подтверждённых реальным темпом обучения.
