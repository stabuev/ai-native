# Артефакт: личная матрица «задача → модель»

Под свои реальные задачи фиксируешь, какую модель берёшь по умолчанию и почему.
Стартовую рекомендацию даёт `recommend()` из [`code/model_map.py`](../code/model_map.py); дальше правишь под себя по опыту.

## Моя матрица (заполнить)

| Моя задача | Профиль (сложность/объём/код?) | Модель по умолчанию | Эскалация на… | Почему |
|---|---|---|---|---|
| Разметка тикетов по категориям | низкая / массовый | Gemini Flash | Haiku при спорных | дёшево на потоке |
| Черновик письма/ответа | низкая / разовый | Claude Haiku 4.5 | Sonnet для важных | быстро и дёшево |
| Разбор требований, план | средняя / разовый | Claude Sonnet 4.6 | Opus на сложном | баланс цена/качество |
| Рефакторинг и ревью кода | высокая / разовый | Claude Opus 4.8 | — | код требует топ-модели |
| _<твоя задача>_ | | | | |

## Как переиспользовать

```python
from model_map import Task, recommend
print(recommend(Task(complexity="low", high_volume=True)))   # → Gemini Flash, …
print(recommend(Task(complexity="high", needs_code=True)))   # → Claude Opus 4.8, …
```

- Начни с рекомендации, замеряй качество на своих кейсах, корректируй столбец «модель по умолчанию».
- Матрица — вход для **Фазы 9 (FinOps)**: там превратим её в автоматический роутер с каскадом и quality-gate.
