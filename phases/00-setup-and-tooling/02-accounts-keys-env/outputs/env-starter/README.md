# env-starter

Drop-in набор для воспроизводимого окружения в любом проекте курса.

## Файлы

- `.env.example` — контракт ключей (без значений). Коммитится.
- `check_env.py` — самодостаточный валидатор (без зависимостей, без сети).

## Использование

```bash
cp .env.example .env          # 1) создать локальный .env
# 2) вписать реальные ключи из консолей Anthropic / OpenAI / Google
echo ".env" >> .gitignore     # 3) НИКОГДА не коммить .env
python check_env.py           # 4) проверить, что всё на месте (exit 0 = ок)
```

## Зачем

Ловит типовые ошибки (нет ключа, остался `your-key-here`, неверный префикс) офлайн —
до первого 401 от API. `check_env.py` можно повесить в CI или pre-commit.
