## Самопроверка перед передачей

- [ ] Задача и критерий сформулированы до интерпретации результата.
- [ ] Между сериями изменилась ровно одна переменная.
- [ ] Указаны входы, число повторов и условия воспроизведения.
- [ ] Наблюдения отделены от причинного объяснения.
- [ ] Решение опирается на результаты, а не на готовый диапазон из чужой шпаргалки.
- [ ] Низкая случайность не выдана за истинность или общее качество.
- [ ] Для API не заявлена гарантия, которой нет в официальном контракте.
