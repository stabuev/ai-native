# Реестр проверки ответа

Заполняй его перед тем, как использовать фактический ответ модели в письме, документе,
решении или автоматизированном процессе. Реестр не доказывает истину вообще: он
показывает, какие утверждения проверены по выбранным источникам и какое решение на этом
основано.

## 1. Границы проверки

- **Задача:** …
- **Ответ или ссылка на него:** …
- **Доверенные источники:** …
- **Разрешены ли внешние знания и поиск:** да / нет; условия: …
- **Кто будет использовать результат:** …
- **Цена существенной ошибки:** низкая / средняя / высокая; почему: …

## 2. Утверждения и доказательства

Статусы:

- **подтверждено** — источник прямо поддерживает утверждение;
- **противоречит** — источник сообщает несовместимый факт;
- **не подтверждено** — доступных данных недостаточно; это не означает «ложно».

Разделяй составные фразы: одно проверяемое утверждение — одна строка.

| № | Атомарное утверждение | Точная цитата и источник либо «опоры нет» | Статус | Цена ошибки | Действие |
|---:|---|---|---|---|---|
| 1 | … | … | … | … | принять / проверить / исправить / отклонить / эскалировать |
| 2 | … | … | … | … | … |
| 3 | … | … | … | … | … |
| 4 | … | … | … | … | … |
| 5 | … | … | … | … | … |

## 3. Решение по ответу целиком

- **Решение:** принять / использовать после исправления / дополнительно проверить /
  отклонить / эскалировать.
- **Обоснование:** …
- **Какие части можно использовать:** …
- **Какие части нельзя использовать и почему:** …
- **Что ещё неизвестно:** …
- **Кому и при каком условии эскалировать:** …

## 4. Самопроверка

- [ ] Все существенные числа, даты, имена, условия, причинные связи и ссылки вынесены в
  отдельные строки.
- [ ] У каждого `подтверждено` есть фрагмент, совпадающий с утверждением по смыслу, а не
  только по словам.
- [ ] Проверены отрицания, единицы, сроки, область применимости и исключения.
- [ ] `Не подтверждено` не перепутано с `противоречит`.
- [ ] Итоговое решение следует из таблицы и учитывает цену ошибки.
- [ ] Ограничения источников и условие эскалации названы явно.

Если хотя бы один пункт не выполнен, исправь реестр и повтори самопроверку.
