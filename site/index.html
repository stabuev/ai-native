Как устроен каждый урок

① Build It

Собираем механизм с нуля на чистом Python (stdlib), запускается офлайн на CPU. Понимаешь, как это работает изнутри.

② Use It

То же самое через реальный инструмент/API — мульти-провайдер: Claude, GPT, Gemini (а где уместно — DeepSeek, open-weight).

③ Ship It

Урок отдаёт артефакт: prompt · skill · agent · MCP — переиспользуемый кусок, который остаётся у тебя.
