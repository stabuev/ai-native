# CLAUDE.md — AI Native Course

Это репозиторий курса **AI Native: From Scratch to Agentic Systems**. Здесь ты помогаешь *разрабатывать* курс: писать уроки, код и артефакты в едином формате.

## Что это за курс

Технический, self-paced, **без видео**. Аудитория умеет немного программировать. Полная программа — в `README.md` (13 фаз, 53 урока). Каждая фаза надстраивается над предыдущей.

## Главный принцип: Build It / Use It / Ship It

Каждый урок сначала строит механизм **с нуля** (Build It, без фреймворков), затем запускает то же через production-инструмент (Use It), и отдаёт **артефакт** (Ship It: prompt · skill · agent · MCP). Студент понимает инструмент, потому что собрал его уменьшенную версию.

## Структура урока (строго)

```
phases/<NN>-<phase-slug>/<NN>-<lesson-slug>/
├── code/        # Build It (с нуля) + Use It (через инструмент) + тесты
├── docs/ru.md   # разбор по 6 шагам
└── outputs/     # артефакт урока
```

Слаги фаз и уроков уже зафиксированы в README (колонка «Папка»). Не переименовывай.

## Цикл из 6 шагов в docs/ru.md

1. **MOTTO** — идея урока в одну строку.
2. **PROBLEM** — конкретная боль.
3. **CONCEPT** — интуиция и диаграммы (ASCII/mermaid).
4. **BUILD IT** — реализация с нуля в `code/`, без внешних фреймворков, запускается на CPU офлайн.
5. **USE IT** — то же самое через реальный инструмент/API. Если нужен платный ключ — пометь `**Requires:**` в начале урока.
6. **SHIP IT** — положи артефакт в `outputs/` и опиши, как его переиспользовать.

## Definition of Done для урока

- `docs/ru.md` заполнен по всем 6 шагам.
- В `code/` есть рабочая реализация и **юнит-тест** (`test_*.py`), который проходит.
- В `outputs/` лежит обещанный артефакт (см. колонку Ship It в README).
- Код запускается на CPU без платных ключей (исключение помечается `**Requires:**`).
- Прогон: `pytest phases/<...>/code` зелёный.

## Как писать новый урок

1. Найди урок в README по id (напр. 2.3) и его папку.
2. Вызови скилл: `/lesson-author 2.3` (см. `.claude/skills/lesson-author`).
3. Заполни Build It реальным кодом + тестом, Use It — примером через инструмент, Ship It — артефактом.
4. Прогони тесты, обнови чек-бокс в `PROGRESS.md`.

## Стиль

- Объяснения на русском, код и идентификаторы на английском.
- Минимум зависимостей; стандартная библиотека по умолчанию.
- Никаких заглушек в коде уроков — только рабочие примеры.
- Образец полностью написанного урока: `phases/01-how-llms-work/01-tokens-and-tokenization/`.
